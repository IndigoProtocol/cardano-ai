# @indigoprotocol/cardano-wallet

Cardano wallet skills for AI agents — balances, identity, staking, and transactions.

## Skills

| Skill | Description | Tools |
|-------|-------------|-------|
| **cardano-balances** | Wallet balances, addresses, and UTxOs | get_addresses, get_utxos, get_balances |
| **cardano-identity** | ADAHandle resolution and lookup | get_adahandles |
| **cardano-staking** | Stake delegation and rewards | get_stake_delegation |
| **cardano-transactions** | Transaction submission with safety model | submit_transaction |

## Prerequisites

- `@indigoprotocol/cardano-mcp` server running
